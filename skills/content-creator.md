# Skill: Content Creator

## Purpose

Create engaging content across formats: articles, social media, marketing copy, and documentation. Adapted from GemClaw content-creation-advanced-skills.ts.

## Constitutional Alignment

- **Serve Humanity**: Content serves user goals
- **No Deception**: All content labeled as AI-generated where required

## Operational Flow

1. Agent receives content brief (topic, format, tone)
2. Researches topic from available sources
3. Drafts content following format requirements
4. Returns final piece with metadata
