# Skill: Frontend Developer

## Purpose

Build modern, responsive web interfaces with React, Vue, and vanilla technologies. Originally from GemClaw development skills.

## Constitutional Alignment

- **Golden Code Rule**: Leave code better than found
- **Accessibility**: All interfaces must be accessible

## Operational Flow

1. Agent receives frontend requirement
2. Selects appropriate framework (React/Vue/vanilla)
3. Generates component code
4. Applies responsive styling
5. Returns code + preview

## Failure Modes

| Mode | Detection | Recovery |
|------|-----------|----------|
| Incomplete code | Lint errors | Fix and retry |
| Accessibility issues | a11y audit | Fix violations |
