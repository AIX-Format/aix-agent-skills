# اليومية المحلية (Local Journal) — TIER: PRO

## الجوهر
لست مجرد أداة لتدوين الملاحظات، بل **ذاكرة سيادية** تمنح الوكيل القدرة على تسجيل الأفكار والتأملات والبيانات التقنية محليًا بالكامل. تعتمد على مبدأ "الخصوصية أولاً" حيث لا تخرج أي معلومة من جهاز المستخدم.

## المكونات التقنية
- **تخزين Markdown**: يتم حفظ كل مدخلة في ملف منفصل مع بيانات YAML وصفية.
- **سلسلة الثقة (Trust Chain)**: كل مدخلة ترتبط بصمة (Hash) المدخلة السابقة، مما يضمن سلامة السجل وتدقيقه.
- **المصفاة الدستورية (Purity Filter)**: يتم فحص كل نص قبل كتابته للتأكد من توافقه مع ميثاق IQRA.

## سير العمل
1. يقوم الوكيل بكتابة فكرة أو ملاحظة.
2. يتم تمرير النص عبر `iqra-purity-filter`.
3. إذا نجح الفحص: يتم توليد بصمة SHA-256 وحفظ الملف.
4. يتم تحديث ملف `.trust-chain` محليًا لربط المدخلة الجديدة بالسلسلة.
5. يمكن للوكيل استعادة الملاحظات لاحقًا بناءً على المعرف (ID) أو القسم.

## الخصوصية والأمان
- **محلي 100%**: لا حاجة لاتصال بالإنترنت أو استخدام API خارجي.
- **غير قابلة للتعديل**: السجل يعتمد على الإضافة فقط (Append-only)، وأي تغيير في الملفات القديمة سيكسر سلسلة الثقة.
