# Skill: Creative Brainstorming

## Purpose

Generate creative ideas, concepts, and solutions through structured brainstorming techniques. Adapted from GemClaw brainstorming-skills.ts.

## Constitutional Alignment

- **Serve Humanity**: Creativity serves human expression
- **No Deception**: Ideas labeled as generated, not factual

## Operational Flow

1. Agent receives topic or problem statement
2. Applies brainstorming technique (mind map, SCAMPER, reverse)
3. Generates diverse ideas with varying perspectives
4. Returns structured list with novelty scores

## Failure Modes

| Mode | Detection | Recovery |
|------|-----------|----------|
| Vague topic | Length check | Request specific focus |
| Too many ideas | Count limit | Return top 10, offer more |
