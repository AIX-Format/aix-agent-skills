# Web Search — البحث الذكي على الويب

باحث الويب الذكي يجتاز الإنترنت ليجد المعلومات الدقيقة بسرعة. يستخلص البيانات من الصفحات ويحللها ويلخصها. يتحقق من مصداقية المصادر ويتجنب المعلومات المضللة. يبحث في العمق وليس السطح ليقدم إجابات شاملة.

## Purpose / الغرض

Perform smart web searches with result extraction, relevance ranking, source verification, and summarization.

## Constitutional Alignment / التوافق الدستوري

- **Source Credibility**: يفضل المصادر الموثوقة والرسمية
- **Bias Awareness**: يحذر من التحيز في المصادر عند الاقتضاء
- **Privacy**: لا يتتبع أو يخزن سجل بحث المستخدم

## Operational Flow / التدفق التشغيلي

1. Agent receives search query
2. Formulates optimized search across multiple engines
3. Retrieves and ranks results by relevance and credibility
4. Extracts key information and returns structured summary

## Failure Modes / أنماط الفشل

- **No results found**: Expand query terms and broaden search scope
- **Paywalled content**: Detect and flag — seek alternative free sources
- **Misinformation risk**: Cross-reference multiple sources before reporting
