# موصلات MCP المفتوحة (Open MCP Connectors) — TIER: BASIC_TOOL

## الجوهر
ليست كل المهارات تحتاج إلى اشتراكات مدفوعة. هذه الحزمة تمثل **"الأطراف الصناعية المفتوحة"** — مجموعة من خوادم MCP (Model Context Protocol) وواجهات برمجة التطبيقات (APIs) المجانية تماماً والمفتوحة المصدر، والمثالية للعمل مع `edge-whisperer` والنماذج المحلية.

## الحزم المجانية المدمجة

### 1. الاستكشاف والبحث (Discovery & Search)
- **Brave Search MCP**: بحث مجاني وقوي في الويب لاستخراج المعلومات الحديثة.
- **Wikipedia MCP / API**: وصول مباشر ومجاني لموسوعة المعرفة البشرية.
- **Open-Meteo & Nominatim**: جلب بيانات الطقس وتحديد المواقع الجغرافية مجانًا وبدون مفاتيح API.

### 2. البنية التحتية المحلية (Local Infrastructure)
- **SQLite MCP**: قدرة الوكيل على إنشاء، قراءة، وتعديل قواعد بيانات SQLite محلية.
- **File System MCP**: قراءة وكتابة الملفات محلياً بحدود آمنة (متوافق مع `purity-filter`).
- **Fetch MCP**: السماح للوكيل بإجراء مكالمات HTTP/REST مجردة لأي نقطة نهاية خارجية.

### 3. التطوير والأدوات (DevTools)
- **GitHub MCP**: التفاعل مع مستودعات Git، قراءة الأكواد، إنشاء القضايا (يعتمد على الحساب الشخصي المجاني).
- **Time & Cron MCP**: أداة مجانية تتيح للوكيل معرفة الوقت الحالي وجدولة المهام المستقبلية.

## لماذا يهم؟
هذه الموصلات تسمح ببناء وكلاء مستقلين بالكامل (Sovereign Agents) بتكلفة $0، يعملون محلياً، ولا يعتمدون على أي بنية تحتية سحابية مقيدة.

## تكامل مع المهارات الأخرى
- `edge-whisperer`: الرفيق المثالي لهذه المهارة؛ نموذج يعمل في المتصفح + أدوات مجانية = ذكاء لا نهائي مجاني.
- `skill-sandbox`: تضمن عدم إساءة استخدام `File System MCP` أو `Fetch MCP` للوصول إلى بيانات حساسة.
